ChatClientRK

Features:

-Real time chatting.

-Login with username.

-UserCommands: /Help, /Users, /History, /Top and /Quit

-Help displays all available commands.

-Users shows all users currently on the server.

-History shows messages sent before joining.

-Top shows amount of messages sent.

-Quit disconnects user from server.


Oo design patters:

-Observer pattern is used.

-MVC is used.

-Singletons are used.

-Adapter is used.
